# Kandyan Homes Website

A modern real estate website built using React and Vite.

## Features
- Modern responsive design
- Property showcase layout
- Fast performance using Vite
- Clean UI inspired by premium real estate websites

## Technologies Used
- React
- Vite
- CSS

## Installation

Clone the repository:

git clone https://github.com/Sahan1221/kandyan-homes-website.git

Install dependencies:

npm install

Run the project:

npm run dev
